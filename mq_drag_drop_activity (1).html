Explanation
Zone 1: Manage Cognitive Load

Cards 2, 5, and 8 all address cognitive load — the amount of mental effort working memory is asked to carry.

Using a worked example (Card 2) reduces the effort needed to understand the task by providing a clear model before independent practice. Sequencing one technique at a time (Card 5) avoids overwhelming working memory with too many new elements at once. Removing the distracting image (Card 8) eliminates extraneous load (cognitive effort that doesn't contribute to learning).

The principle: reduce what isn't necessary so working memory can focus on what matters.

Zone 2: Ensure Deep Thinking

Cards 3, 6, and 9 all require students to engage elaboratively by extending, connecting, or evaluating ideas rather than passively receiving them.

Analysing a persuasive text (Card 3) requires students to interpret meaning and technique — not just identify. Comparing techniques with a partner (Card 6) builds connections between new and prior knowledge, which deepens encoding. Drafting and comparing against a model (Card 9) requires self-evaluation (a metacognitive and elaborative process).

The principle: deep encoding happens when students do the thinking, not when they watch or listen.

Zone 3: Retrieval Practice

Cards 1, 4, and 7 all ask students to actively pull information from long-term memory back into working memory.

The opening recall prompt (Card 1) is a classic retrieval practice task (free recall before the lesson begins), which also strengthens the memory pathway for prior content. The end-of-lesson brain dump (Card 4) consolidates the session's learning through self-generated retrieval. The opening discussion about persuasion (Card 7) activates relevant schemas and prior experience, which is a form of retrieval that also prepares long-term memory to receive new information.

The principle: every time students retrieve knowledge, that memory pathway is strengthened. Retrieval practice doesn't have to look like a test.
